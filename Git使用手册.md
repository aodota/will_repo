Git使用手册
===========

### 下载Git Windows
[Git Windows](https://git-scm.com/download/)

### Git设置

- 在本地新建Git Workspace
- 进入该目录，右键`Git Bash Here`
- 输入命令

```Bash
# 初始化git
git init

# 存储git用户名和密码
git config --global credential.helper store
```
- 修改Git配置
